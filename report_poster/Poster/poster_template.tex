\begin{minipage}[t]{0.48\textwidth}\vspace{0pt}

  % -- Objectives ----
  \begin{tcolorbox}[posterbox, title=Objectives]
    \begin{itemize}
    \item Recommend new music by enriching collaborative filtering (CF) with \textbf{symbolic + audio content} — genre, artist links, instrumentation, tempo.
    \item Encode it as a typed \textbf{Knowledge Graph} relating users, tracks, artists and genres beyond a user–item matrix.
    \item Learn over the KG with a \textbf{Heterogeneous Graph Transformer (HGT)} for diversity, cold-start robustness and \textbf{explainability}.
    \end{itemize}
  \end{tcolorbox}

  % -- Problem ----
  \begin{tcolorbox}[posterbox, title=Problem]
    \begin{itemize}
      \item CF uses only interaction history → \textbf{popularity bias} and \textbf{cold-start} failures, ignoring musical content.
      \item Most KG-enhanced work uses shallow embeddings~[8,\,10], without a GNN that propagates \textbf{multi-hop typed} structure.
      \item \textit{Can symbolic features + a heterogeneous KG + an HGT reason over shared genre / artist / instrumentation to close that gap?}
    \end{itemize}
  \end{tcolorbox}

  % -- Methodology ----
  \begin{tcolorbox}[posterbox, title=Methodology]
    \begin{center}
      \includegraphics[width=0.98\linewidth]{methodology.png}
    \end{center}
    % {\small \textbf{HGT}~[1] message-passing over the undirected KG; $\ell_2$-normalised embeddings $\to$ \textbf{cosine} scoring. Trained with a \textbf{custom listwise loss} (\emph{not} pointwise binary cross-entropy nor pairwise BPR~[11]): a soft cross-entropy over each user's candidate softmax, logit-adjusted to down-weight popular tracks:}
    % {\small
    % \[ s_i = \tfrac{\hat{\mathbf{u}}^\top\hat{\mathbf{t}}_i}{\tau} + \lambda\log p_i,\qquad
    %    \mathcal{L} = -\tfrac{1}{|B|}\sum_{u}\sum_i \tilde{p}_{ui}\,\log\mathrm{softmax}(\mathbf{s}_u)_i \]}
    % {\small $\tau{=}0.1$; soft targets $\tilde{p}_{ui}\!\propto\!\log(1{+}c_{ui})$. As $\log p_i<0$, the $\lambda\log p_i$ term \textbf{boosts less-heard tracks} — driving the low popularity bias below.}
    Model selection accounts for relevance, diversity \& fairness by using: \\\textbf{Overall Score@10}$=0.6 NDCG + 0.2 Cov + 0.2 (1 - PopularityBias)$
  \end{tcolorbox}
  
  \begin{tcolorbox}[posterbox, title=Recommendation Benchmark]
    \begin{center}
      \includegraphics[width=\linewidth]{benchmark_heatmap_poster.png}\\[4pt]
      % \includegraphics[width=0.78\linewidth]{benchmark_significance_poster.png}\\[2pt]
      % \fcap{Lighter = better. Violins = per-user NDCG distribution; bars = set-level Overall Score $\pm$ bootstrap std; $\ast$ = significant.}
    \end{center}
    \begin{itemize}
    \item HGT trails KNN-CF for short lists but \textbf{overtakes it for $K\geq 50$}: Overall Score \textbf{0.474 vs 0.455} and NDCG \textbf{0.148 vs 0.112} at $K{=}100$, with the \textbf{lowest popularity bias (0.090)} throughout.
    % \item MostPopular's high raw NDCG (0.114) is a \textbf{popularity artifact}: near-zero coverage (0.006), extreme bias (0.611) → composite 0.148.
    \end{itemize}
  \end{tcolorbox}
  
  % -- Baselines ----
  \begin{tcolorbox}[posterbox, title=Baselines]
    % {\small\itshape each: features $\cdot$ preprocessing $\cdot$ objective $\to$ output}
    \begin{itemize}
    \item \textbf{MostPopular} — global play counts, no user features; rank by frequency $\to$ one list shared by all users.
    \item \textbf{KNN-CF} — binarised, $\ell_2$-normalised interactions; user–user cosine $k$NN ($k{=}6$) $\to$ neighbour-weighted scores.
    \item \textbf{XGBoost-Hybrid} — AE user+track embeddings $+$ 44 categorical (1-hot key/mode/tempo; target-enc.\ genre/artist; multi-hot instrument); 200 AE-cosine candidates/user, gradient-boosted LTR re-rank (\texttt{rank:ndcg@10}) $\to$ top-$K$.
    \end{itemize}
  \end{tcolorbox}

  \begin{tcolorbox}[posterbox, title=Explainability]
    \begin{center}
      \includegraphics[width=\linewidth]{explain_attention_poster.png}\\[2pt]
      \fcap{Left edge: Each track's attention share (top 5 of 134). Right edge: Share of attention sitting on tracks carrying the attribute.\\Faithful attention traces users {\large$\to$} tracks {\large$\to$} shared attributes {\large$\to$} recs.}
    \end{center}
    \begin{itemize}
    \item Captured HGT \textbf{attention} (not a post-hoc proxy) explains \emph{why} each track was recommended, attribute by attribute.
    \item Cross-checked against \textbf{occlusion $\Delta$score} and \textbf{Integrated Gradients} — the three agree on the top drivers.
    \end{itemize}
  \end{tcolorbox}

  % -- Conclusions ----
  \begin{tcolorbox}[posterbox, title=Conclusions \& Future Work]
    \begin{itemize}
      \item The HGT \textbf{matches and overtakes} the strongest CF baseline at depth ($K\geq50$) on NDCG, coverage \& Overall Score, with the \textbf{lowest popularity bias} of all recommenders.
      \item Symbolic + audio content in a \textbf{typed KG} gives interpretable, diverse recommendations; brand-new users are served through \textbf{listener archetypes} (cold-start).
      \item Captured \textbf{HGT attention} yields faithful, attribute-level explanations, corroborated by occlusion $\Delta$score and Integrated Gradients.
    \end{itemize}
    \vspace{2pt}{\bfseries\color{sectionfg} Future work}
    \begin{itemize}
      \item \textbf{Adaptive popularity control}: learn / anneal the debiasing weight $\lambda$ and temperature $\tau$ to trade relevance against fairness on demand.
      \item A \textbf{hybrid HGT\,+\,CF} re-ranker that exploits the depth crossover — CF for short lists, the HGT for long, diverse ones.
      \item \textbf{Richer, multi-hop KG signals} (lyrics, mood, social ties) and deeper propagation beyond the single user$\to$track hop.
      \item \textbf{Temporal / session-aware} modelling, plus a human study of how useful the attention explanations are to listeners.
    \end{itemize}
  \end{tcolorbox}
